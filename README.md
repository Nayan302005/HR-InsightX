📌 Overview
The HR Analytics Dashboard is a powerful Power BI project designed to analyze workforce data, uncover employee trends, and support data-driven HR decisions.

This dashboard provides deep insights into employee attrition, demographics, salary distribution, and job roles, helping organizations optimize their workforce strategy.
